# 囚徒困境 | Prisoner's Dilemma

> 用 3 分鐘理解賽局理論的經典概念

## 📖 什麼是囚徒困境？

囚徒困境是賽局理論中最著名的思想實驗，揭示了「個人理性」與「集體利益」之間的矛盾。

### 故事背景

你和另一名嫌犯被警方逮捕並分開審訊。警方提出交易：

- **雙方都沉默（合作）**：各判 1 年
- **一方背叛，一方沉默**：背叛者釋放，沉默者判 3 年
- **雙方都背叛**：各判 2 年

### 核心矛盾

雖然「雙方都合作」對整體最有利（總刑期 2 年），但從個人角度，「背叛」永遠是更理性的選擇：

- 如果對方合作，我背叛 → 我 0 年（比合作的 1 年好）
- 如果對方背叛，我背叛 → 我 2 年（比合作的 3 年好）

結果：雙方都選擇背叛，陷入「雙輸」局面（總刑期 4 年）。

## 🤖 AI 策略：Tit-for-Tat

本應用的 AI 採用經典的「以牙還牙」策略：

- **第 1 輪**：隨機選擇
- **第 2-5 輪**：模仿你上一輪的選擇

這個策略在 Robert Axelrod 的「重複囚徒困境錦標賽」中證明是最有效的策略之一。

## 💡 現實應用

囚徒困境不只是理論，它在現實生活中無處不在：

### 商業競爭

- 兩家公司可以選擇「價格戰」（背叛）或「維持價格」（合作）
- 雙方都打價格戰 → 利潤都下降（雙輸）

### 環境保護

- 每個國家可以選擇「減排」（合作）或「繼續污染」（背叛）
- 如果大家都不減排 → 氣候災難（雙輸）

### 團隊合作

- 每個成員可以選擇「努力工作」（合作）或「偷懶」（背叛）
- 如果大家都偷懶 → 專案失敗（雙輸）

## 🧠 思考問題

1. 為什麼「雙方都背叛」是最常見的結果，即使「雙方都合作」對大家更好？
2. 如果可以溝通，結果會改變嗎？
3. 在你的生活中，你遇過類似的情境嗎？
